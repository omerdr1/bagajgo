# BagajGo - Akıllı Bagaj Yönetim Platformu

BagajGo, Türkiye ve KKTC'de seyahat edenlerin bagaj taşıma sorununu çözen akıllı bir bagaj yönetim platformudur. Platform, bagaj depolama, şehir içi teslimat, şehirlerarası transfer ve gerçek zamanlı takip gibi hizmetler sunmaktadır.

## Özellikler

- 🎒 Bagaj Depolama
- 🚚 Şehir İçi Teslimat
- ✈️ Şehirlerarası Transfer
- 📍 Emanet Noktası Eşleştirme
- 🗺️ Rota Optimizasyonu
- 📱 Gerçek Zamanlı Takip
- 🔒 Güvenlik Sistemleri
- 💳 Temassız Ödeme
- 👤 Kullanıcı Yönetimi
- 📊 Yönetici Paneli

## Teknolojiler

- CSS3
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts
